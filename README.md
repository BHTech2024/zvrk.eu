# zvrk.eu
Dobrodošli na Zvrk.eu – prvi balkanski portal posvećen dijeljenju kratkih video sadržaja. Na Zvrk.eu korisnici mogu uploadati, pregledavati i komentarisati kratke video zapise, pratiti najnovije trendove i interaktivno učestvovati u zajednici.
